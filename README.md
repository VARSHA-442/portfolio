# Stock Portfolio Tracker

## Overview

The **Stock Portfolio Tracker** is a user-friendly web application designed for tracking and managing stock portfolios. It allows users to:
- Add and edit stock details.
- View their current stock holdings.
- Analyze portfolio performance with a dashboard.

---

## Features

### 1. Dashboard
- **Total Portfolio Value**: Displays the total value of your holdings.
- **Top-Performing Stock**: Identifies the stock with the highest value.

### 2. Add/Edit Stock
- Easily add new stocks with:
  - **Stock Name** 
  - **Ticker**
  - **Quantity**
  - **Buy Price**
- Edit existing stock details.

### 3. Current Holdings
- View all stocks in a table with details:
  - Stock Name
  - Ticker Symbol
  - Quantity
  - Buy Price
- Manage your holdings with options to **Edit** or **Delete** stocks.

### 4. Stock Tables
- **Stocks to Buy Today**:
  - Displays recommended stocks with key metrics like CMP, SL, and targets.
- **Stocks at Loss**:
  - Lists underperforming stocks with detailed performance metrics.

---

## Technologies Used

### Frontend
- **HTML**: Provides the structure for the web app.
- **CSS**: Custom styles to create a clean and responsive UI.
- **JavaScript**: Enables dynamic behavior, such as adding/editing stocks and updating the dashboard in real time.
